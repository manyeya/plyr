# Plyr

A feature-rich media player (audio & video) embedded directly inside VS Code. Listen to your favorite tracks or watch videos without leaving your editor!

## Features

- **Embedded Playback**: Watch videos or listen to audio directly from the VS Code sidebar or panel.
- **Playlist Management**: Easily manage your now playing queue.
- **Controls**: Play, pause, skip, and adjust volume or playback speed right from the VS Code interface.
- **Keyboard Shortcuts**: Quickly toggle playback using standard shortcuts.

## Extension Commands

- `Plyr: Open Player` - Opens the media player view.
- `Plyr: Toggle Play/Pause` - Toggles the current media playback.
- `Plyr: Open Media File` - Opens a local media file to play.
- `Plyr: Next Track` - Skips to the next track.
- `Plyr: Previous Track` - Goes back to the previous track.

## Extension Settings

This extension contributes the following settings:

* `mediaPlayer.defaultVolume`: Default playback volume (0-100).
* `mediaPlayer.autoplay`: Automatically play media when opened.
* `mediaPlayer.defaultSpeed`: Default playback speed (0.5x to 2.0x).
